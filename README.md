# Fitness-Einstellungstool

Eine kleine interaktive Webseite fuer ein biologisch plausibles Fitness-Recomp-Modell.

## Funktionen

- Regler fuer Fettabbau, Muskelaufbau, Anstrengung und Hunger
- Automatische Kopplung der Regler nach Recomp-Logik
- Live-Auswertung mit Strategie, Kalorienlage, Trainingsreiz und Erholungsdruck
- Laeuft als statische Webseite ohne Build-Schritt

## Lokal starten

Oeffne `index.html` direkt im Browser.

## Quellen fuer die Modellannahmen

- ISSN Position Stand: Diets and Body Composition
- Body Recomposition Ueberblick von Healthline
